Branded QR State Test

Open the browser console and navigate to the admin dashboard to see debug logs.

Instructions:

  1. Open browser developer tools (F12)
  2. Go to the Console tab
  3. Navigate to http://localhost:3001/admin/dashboard
  4. Look for debug logs:
    • "AdminDashboard mounted, showBrandedQRModal: false"
    • When clicking Branded QR: "onShowBrandedQR called with: {campaign data}"
    • "showBrandedQRModal set to true"

If you still see the error:

  1. Clear browser cache: Ctrl+Shift+R (hard refresh)
  2. Clear all browser data: Go to browser settings and clear all data for localhost
  3. Try incognito/private mode
  4. Check if multiple tabs are open - close all localhost tabs and try again
