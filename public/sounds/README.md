# Sound Files

Place the following sound files in this directory:

- `spin.mp3` - Wheel spinning sound (looping)
- `tick.mp3` - Quick tick sound for each segment pass
- `win.mp3` - Victory sound when prize is won

## Free Sound Resources

- https://freesound.org
- https://mixkit.co/free-sound-effects/

## Recommended Sounds

- **spin.mp3**: A continuous spinning/whirring sound (2-3 seconds, will loop)
- **tick.mp3**: A short click/tick sound (< 0.5 seconds)
- **win.mp3**: A celebratory fanfare or success sound (1-2 seconds)

## File Format

All files should be in MP3 format for maximum browser compatibility.
